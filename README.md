# Personal Portfolio Site

This repository contains the source code for my personal portfolio website, built with Next.js. It's super old, but gets the job done, and serves a central hub for my professional work, including projects, experience, and contact information. Coming back in 2026, hoping to push out a whole new ground up site soon, just need to get a couple of projects I've been working on finished up first. 

You can view the live site here: <https://willvarner.me>


## Technologies Used

* **Framework**: Next.js

* **Languages**: TypeScript, JavaScript, HTML, CSS

* **Styling**: Tailwind CSS

* **Deployment**: Github Pages
